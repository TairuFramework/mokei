# MCP draft groundwork G5 + G8 — completed

**Status:** complete
**Date:** 2026-06-10

## Goal

Land the two now-unblocked MCP-draft Phase 0 groundwork items — JSON Schema 2020-12
tool/prompt schemas (G8, SEP-2106) and W3C trace-context propagation in request `_meta`
(G5 outbound, SEP-414) — additively on the `2025-11-25` baseline, and drop ephemeral
plan-label references (G1/G4/G6) from existing test names per the conventions update.

## What was built

- **G8 — opt-in 2020-12 schema validation.** `inferSchemaDraft(schema)` in
  `context-protocol` reads the `$schema` dialect URI and returns `'2020-12'` when it
  declares that dialect, else `'07'`. `context-server`'s `createTool`/`createPrompt` pass
  the inferred draft into `@enkaku/schema@0.16.1`'s `{ draft }` option (lazy `Ajv2020`).
- **G5 outbound — trace context in `_meta`.** New `context-client/src/trace.ts` builds a
  `TraceMeta` from the active OTel span (`@enkaku/otel@0.16.1`: `getActiveTraceContext` for
  `traceparent`, `getActiveSpan().spanContext().traceState.serialize()` for `tracestate`).
  A `ContextClient.request()` override merges `{traceparent, tracestate}` into
  `params._meta`. Added `@enkaku/otel` to the workspace catalog + `context-client` deps.
- **Conventions:** removed `(G1)`/`(G4)/(G6)/(G1 server)` labels from `describe` names in
  the two test files.
- **Docs:** milestone Phase 0 table marks G5 (outbound) + G8 done; backlog + roadmap
  rewritten to remaining-only.

## Key design decisions

- **Both items are non-breaking by construction.** G8 defaults to draft-07, so schemas
  without `$schema` (every existing one) validate byte-for-byte unchanged — only schemas
  that opt in via the 2020-12 dialect get the relaxed keywords. G5 injection is a zero-cost
  early-return when no OTel SDK is active (`getActiveTraceContext()` → `undefined`).
- **Single G5 injection point.** Overriding `ContextRPC.request()` in `ContextClient`
  covers every client request method in one place, and lives in a vitest-capable package
  (`context-rpc` has no unit-test runner).
- **Flat W3C `_meta` keys**, not DNS-prefixed — SEP-414's explicit exception to the
  `_meta` key convention; `2025-11-25` peers ignore the extra keys.
