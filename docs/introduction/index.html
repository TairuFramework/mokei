Mokei MCP toolkit

Mokei is a toolkit for creating, interacting and monitoring clients and servers using the Model Context Protocol (MCP).

What is MCP?​

The Model Context Protocol (MCP) is a protocol specifying interactions between Large Language Models (LLMs) and external systems to enhance the possibilities of LLM interactions.

Why MCP?​

The Model Context Protocol is an open standard created by Anthropic and supported by their Claude for Desktop application. As such, it has the backing of a major AI company while being fully free to use by anyone.

Using MCP rather than proprietary alternatives allow developers to create solutions that can integrate with a wider ecosystem without particular restrictions, while benefitting from the stability of a well-defined standard.

Why Mokei?​

Easily create MCP servers, clients and hosts​

The Mokei libraries help create MCP servers and clients, as well as hosts spawning servers and managing their lifecycle.

Proxy and monitor client-server interactions​

The Mokei Monitor is a Web UI running locally that allows you to monitor the interactions between your MCP clients and servers using a local proxy.

Interact with models running locally​

The Mokei CLI provides a simple integration with Ollama, allowing to provide MCP servers to conversations with models running locally.

Get started now​

Follow the quick start guide to learn how to create your first MCP server and interact with it.
