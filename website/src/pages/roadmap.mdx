# Roadmap

Mokei aims to provide a reliable set of tools for developers to easily create, test and maintain MCP servers, clients and hosts.

Features development milestones can be [tracked on GitHub](https://github.com/TairuFramework/mokei/milestones), while this page provides a high-level overview of the roadmap.

## Milestones

### Full protocol support

[GitHub milestone](https://github.com/TairuFramework/mokei/milestone/1)

The goal of this milestone is to implement full support of the [first version (2024-11-05) of the MCP protocol](https://spec.modelcontextprotocol.io/specification/2024-11-05/) in the Mokei client and server packages.

### SSE transport support

[GitHub milestone](https://github.com/TairuFramework/mokei/milestone/3)

The goal of this milestone is to implement support for the [Server-Sent Events (SSE) transport](https://spec.modelcontextprotocol.io/specification/2024-11-05/basic/transports/#http-with-sse) in the Mokei client and server packages, the host server and the CLI.

### Alternative LLM providers support

[GitHub milestone](https://github.com/TairuFramework/mokei/milestone/2)

The goal of this milestone is to add support for other LLM providers than Ollama in the Mokei CLI, starting by the OpenAI API.
