# Mokei APIs

## Context

These packages implement the Model Context Protocol client and server.

- [`@mokei/context-protocol`](./api/context-protocol/index.md): MCP constants, schemas and types.
- [`@mokei/context-rpc`](./api/context-rpc/index.md): RPC for MCP, used by the client and server.
- [`@mokei/context-server`](./api/context-server/index.md): MCP server.
- [`@mokei/context-client`](./api/context-client/index.md): MCP client.

## Host

These packages support managing and monitoring MCP servers and clients.

- [`@mokei/host-protocol`](./api/host-protocol/index.md): Host protocol constants, schemas and types.
- [`@mokei/host`](./api/host/index.md): Host server and client.
- [`@mokei/host-monitor`](./api/host-monitor/index.md): Host monitor UI server.

## Session

High-level abstractions for chat sessions with MCP integration.

- `@mokei/session`: Session management combining hosts with model providers.

## Model Providers

These packages provide integrations with AI model providers.

- `@mokei/model-provider`: Base types and interfaces for model providers.
- `@mokei/openai-provider`: OpenAI API integration.
- `@mokei/ollama-provider`: Ollama local models integration.
